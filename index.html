CS 499 Milestone Two

Enhancement Narrative

Artifact Description

The artifact selected for this category is the Animal Shelter CRUD application originally developed in CS 340: Advanced Programming Concepts. It is a Python-based web dashboard built with the Dash framework that interfaces with a MongoDB database to perform create, read, update, and delete operations on animal shelter data for the fictional client Grazioso Salvare.

Justification & Improvements

While functional, the original artifact lacked any authentication, input validation, or data protection mechanisms. I selected it to demonstrate my ability to take a prototype and elevate it to production-grade security standards.

The software engineering enhancement focused on adding security hardening to the application layer. This included implementing user authentication via hashed credentials (bcrypt), input validation on all user-supplied fields to prevent injection attacks, and encrypting sensitive data fields at rest. These enhancements showcase my skills in secure software design, defensive programming, and the application of cryptographic principles.

Course Outcomes Alignment

Through this enhancement, I successfully met the planned course outcomes by demonstrating the ability to use well-founded techniques in computing practices. By implementing proactive security measures against injection and authentication bypass attacks, I illustrated substantial progress toward developing a security mindset that anticipates adversarial exploits and mitigates design flaws.

Reflection & Challenges

The process of enhancing this artifact reinforced the critical importance of a defense-in-depth approach. One of the primary challenges was configuring the bcrypt library in Python and ensuring that password hashing integrated smoothly with the existing Dash callbacks without breaking the user experience. By resolving these issues, I deepened my practical understanding of integrating robust security mechanisms into existing codebases, directly aligning with my career goals as a secure software engineer.
